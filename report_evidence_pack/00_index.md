# Evidence Pack Index — Euda App / Agentic Coding Case Study

## Executive Framing

This evidence pack documents the development of **Euda** — a social event-discovery and check-in app for iOS and Android — as a primary applied case for an innovation report on **agentic coding systems**. Euda was built by a single founder using **Claude Code** (Anthropic's terminal-native agentic coding agent) as the primary development tool. The entire application — from initial commit to App Store readiness — was produced in **38 calendar days** (January 17 – February 23, 2026) across **56 git commits**, encompassing a full-stack mobile app, 108 PostgreSQL migrations, 20 Supabase edge functions, a 9-signal recommender engine, content moderation, push notifications, and an admin dashboard.

The documents below were generated by systematic analysis of the live codebase, git history, CI configuration, documentation, and dependency manifests. Where data is estimated rather than measured, it is explicitly labeled as such. External claims include citation notes.
