\# EUDA V1 Roadmap

\- Expo app runs on iPhone

\- Supabase connected

\- events table exists and seeded

\- App can fetch and display events



\## Phase 1 (now)

\- profiles table + RLS

\- Create profile row on signup

\## Phase 2

\- Add 3-tab navigation: Feed / Explore / Profile

\- Explore tab lists events + activities (single list)

\- "I'm going" (RSVP)

\- Check-in record created when in range

\- Camera-only post flow:

&nbsp; - front / back / dual

&nbsp; - no uploads, no filters

\- Save photos to Supabase Storage + metadata in DB

\- Feed shows friends posts chronologically



\## Phase 4

\- Friend requests + accept/decline

\- Likes + comments (basic)
